An extremely lightweight World of Warcraft addon designed to make your in-game cursor much easier to locate during intense combat or chaotic encounters. 

*✨ Never lose track of your mouse pointer again! ✨*

---

## 🚀 Features

* **Dynamic Scaling:** Displays a star-like tracking symbol behind your mouse pointer that adapts its size dynamically depending on how fast you move your cursor.
* **Hyper-Lightweight:** Built from the ground up to be future-proof and highly optimized, ensuring zero impact on your framerate (FPS).
* **Cross-Flavor Support:** Fully compatible with both **Retail** and **Classic** flavors of World of Warcraft.

## ⚙️ Configuration & Commands

You can access the configuration options to adjust the behavior of the tracker using the game interface or via chat:

* **In-Game Menu:** Press `Esc` ➡️ `Options` ➡️ `AddOns` ➡️ **StarCursor**
* **Slash Command:** Type `/starcursor` in your chat bar.

## 📦 Installation

To ensure you always have the latest, verified, and secure version of StarCursor, all downloads are hosted exclusively on CurseForge.

### Option 1: Automatic Update (Recommended)
The easiest way to install and keep StarCursor updated automatically is through the CurseForge App:
👉 **[Download StarCursor on CurseForge](https://www.curseforge.com/wow/addons/starcursor)**

### Option 2: Manual Installation
If you prefer to install your addons manually, please download the official release package directly from CurseForge to ensure you get the latest stable build:
1. Go to the **[StarCursor CurseForge Page](https://www.curseforge.com/wow/addons/starcursor)** and click **Download**.
2. Extract the contents of the downloaded `.zip` archive.
3. Move the extracted `StarCursor` folder into your World of Warcraft AddOns directory:
   * **Retail:** `World of Warcraft/_retail_/Interface/AddOns/`
   * **Classic:** `World of Warcraft/_classic_/Interface/AddOns/`
4. Restart or reload the game (`/reload`).
